Frequently Asked Questions

Is it safe to download?

Yes! We're verified developers. The .crx file is cryptographically signed by Google.

Will it slow down Chrome?

No. The extension only runs when you activate it with Ctrl+Shift+S. Minimal memory usage.

Can I uninstall it?

Yes. Go to chrome://extensions, find "Naraseo AI", click the trash icon. Uninstalls completely.

What if drag-drop doesn't work?

Try this alternative:
1. Open chrome://extensions
2. Toggle "Developer mode" (top right)
3. Click "Load unpacked"
4. Select the extracted extension folder
