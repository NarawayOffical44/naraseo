API Reference
Quick reference for the most commonly used endpoints.
Base URL: https://naraseoai.onrender.com  —  Auth: Authorization: Bearer {your_token}  —  OpenAPI spec →
Authentication
POST /api/auth/signup
Create a new account. Body: { name, email, password }. Returns { token, user }.
POST /api/auth/login
Authenticate. Body: { email, password }. Returns { token, user }.
GET /api/auth/me
Get current user profile. Requires Bearer token. Returns { id, name, email, plan }.
SEO Endpoints
POST /api/v1/audit
Full SEO audit. Body: { url }. Returns score, grade, issues, keywords, Core Web Vitals.
POST /api/v1/solve
Autonomous audit + fix. Body: { url }. Returns audit data plus AI-generated fixes.
POST /api/v1/solve-site
Site-wide autonomous SEO via sitemap. Body: { url }. Pro+ plan required.
POST /api/v1/competitors
Competitor gap analysis. Body: { url, competitors: [] }. Uses real domain authority data.
POST /api/v1/report
Generate PDF report. Body: { url }. Returns PDF binary (Content-Type: application/pdf).
GET /api/v1/geo-grid/history
Geo-grid rank history. Query: ?url=&keyword=. Pro+ plan required.
MCP Server
POST /mcp
Streamable HTTP MCP transport. Use with Claude, Cursor, or Windsurf.
GET /mcp/sse
SSE legacy MCP transport. Use with npx mcp-remote https://HOST/mcp/sse.
GET /mcp
Returns full MCP connection guide JSON with all 10 tool definitions.
