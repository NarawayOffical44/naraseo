1. Get API Key

Code Examples

Python

import requests import json def audit_page(url, api_key): response = requests.post( 'https://naraseoai.onrender.com/api/v1/audit', headers={'Authorization': f'Bearer {api_key}'}, json={'url': url} ) return response.json() result = audit_page('https://example.com', 'YOUR_API_KEY') data = result['data'] print(f"URL: {data['url']}") print(f"Score: {data['score']}/100") print(f"Grade: {data['grade']}") print(f"\nIssues ({len(data['issues'])}):") for issue in data['issues']: print(f" [{issue['type'].upper()}] {issue['title']}") print(f" → {issue['suggestion']}") print(f"\nCore Web Vitals:") print(f" LCP: {data['coreWebVitals']['lcp']}s") print(f" FID: {data['coreWebVitals']['fid']}ms") print(f" CLS: {data['coreWebVitals']['cls']}")

JavaScript

async function auditPage(url, apiKey) { const response = await fetch('https://naraseoai.onrender.com/api/v1/audit', { method: 'POST', headers: { 'Authorization': `Bearer ${apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ url }) }); const result = await response.json(); const data = result.data; console.log(`URL: ${data.url}`); console.log(`Score: ${data.score}/100`); console.log(`Grade: ${data.grade}`); console.log(`\nIssues (${data.issues.length}):`); data.issues.forEach(issue => { console.log(` [${issue.type.toUpperCase()}] ${issue.title}`); console.log(` → ${issue.suggestion}`); }); console.log('\nCore Web Vitals:'); console.log(` LCP: ${data.coreWebVitals.lcp}s`); console.log(` FID: ${data.coreWebVitals.fid}ms`); console.log(` CLS: ${data.coreWebVitals.cls}`); } auditPage('https://example.com', 'YOUR_API_KEY');

Node.js with Express

const express = require('express'); const axios = require('axios'); const app = express(); const API_KEY = process.env.NARASEO_API_KEY; app.post('/api/audit', async (req, res) => { try { const { url } = req.body; const response = await axios.post( 'https://naraseoai.onrender.com/api/v1/audit', { url }, { headers: { 'Authorization': `Bearer ${API_KEY}`, 'Content-Type': 'application/json' } } ); const data = response.data.data; res.json({ success: true, score: data.score, grade: data.grade, issues: data.issues.filter(i => i.type === 'critical'), cwv: data.coreWebVitals }); } catch (error) { res.status(500).json({ error: error.message }); } }); app.listen(3000, () => console.log('Running on port 3000'));

Python Example

import requests def analyze_keywords(url, api_key): response = requests.post( 'https://naraseoai.onrender.com/api/v1/keywords', headers={'Authorization': f'Bearer {api_key}'}, json={'url': url} ) return response.json()['data'] keywords = analyze_keywords('https://example.com', 'YOUR_API_KEY') print(f"Primary Keyword: {keywords['aiAnalysis']['primaryKeyword']}") print(f"Search Intent: {keywords['aiAnalysis']['searchIntent']}") print(f"\nQuick Wins:") for win in keywords['aiAnalysis']['quickWins']: print(f" • {win['keyword']} - {win['action']}") print(f"\nContent Gaps: {', '.join(keywords['aiAnalysis']['contentGaps'])}")

async function downloadReport(url, apiKey) { const response = await fetch('https://naraseoai.onrender.com/api/v1/report', { method: 'POST', headers: { 'Authorization': `Bearer ${apiKey}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ url }) }); const blob = await response.blob(); const downloadUrl = window.URL.createObjectURL(blob); const a = document.createElement('a'); a.href = downloadUrl; a.download = `seo-report-${new Date().getTime()}.pdf`; a.click(); window.URL.revokeObjectURL(downloadUrl); } downloadReport('https://example.com', 'YOUR_API_KEY');

MCP Server Integration

Give Claude, Cursor, and Windsurf native SEO tools

What is MCP?

Model Context Protocol allows AI agents to call tools directly. Instead of making HTTP requests, Claude can use SEO tools natively.

Setup for Claude Desktop

  1. Open %APPDATA%\Claude\claude_desktop_config.json
  2. Add this configuration:
{ "mcpServers": { "naraseo-ai": { "command": "npx", "args": ["mcp-remote", "https://naraseoai.onrender.com/mcp/sse"] } } }
  1. Restart Claude Desktop
  2. Ask Claude: "What are the SEO issues on example.com?"

Available MCP Tools

  • seo_audit — Full audit with score and issues
  • solve — Autonomous audit + keywords + fixes
  • keyword_research — Keyword analysis and gaps
  • schema_validate — JSON-LD validation
  • site_crawl — Multi-page crawl
  • geo_grid — Local rank tracking
  • competitor_analysis — Gap analysis with DA

Error Handling

Handle API errors gracefully

HTTP Status Codes

Code Meaning Solution
200 Success Parse the response
400 Bad Request Check URL format and required fields
401 Unauthorized Verify API key is correct and included in header
403 Forbidden Feature not available on your plan
429 Rate Limited Wait before next request or upgrade plan
502 Bad Gateway URL unreachable or returns invalid HTML
500 Server Error Try again in a moment, contact support

Error Response Format

{ "success": false, "error": { "code": "FEATURE_UNAVAILABLE", "message": "Feature 'crawl' not available on Free tier", "requiredTier": "Pro" } }

Python Error Handling

import requests try: response = requests.post( 'https://naraseoai.onrender.com/api/v1/audit', headers={'Authorization': f'Bearer {api_key}'}, json={'url': url}, timeout=30 ) response.raise_for_status() data = response.json() if not data.get('success'): error = data.get('error', {}) print(f"API Error: {error.get('code')} - {error.get('message')}") return print(f"Score: {data['data']['score']}") except requests.exceptions.Timeout: print("Request timeout - try again") except requests.exceptions.ConnectionError: print("Cannot connect to API server") except requests.exceptions.HTTPError as e: if e.response.status_code == 429: print("Rate limited - upgrade plan") elif e.response.status_code == 401: print("Invalid API key") else: print(f"HTTP Error {e.response.status_code}")

Rate Limits & Quotas

Fair usage limits to ensure platform stability

Per-Tier Limits

Plan Per Minute Per Day
Free 10 requests 100 requests
Pro 60 requests 1,000 requests
Agency 200 requests Unlimited

Response Headers

Every response includes rate limit info:

X-RateLimit-Limit: 10 X-RateLimit-Remaining: 7 X-RateLimit-Reset: 1234567890

What Counts Toward Limits

  • Each API call = 1 request (regardless of what endpoint)
  • Site crawl with 50 pages = 50 requests
  • Geo-grid with 9 points = 9 requests

Handling Rate Limits (Python)

import requests import time def audit_with_retry(url, api_key, max_retries=3): for attempt in range(max_retries): response = requests.post( 'https://naraseoai.onrender.com/api/v1/audit', headers={'Authorization': f'Bearer {api_key}'}, json={'url': url} ) if response.status_code == 429: # Rate limited - wait and retry wait_time = int(response.headers.get('X-RateLimit-Reset', time.time() + 60)) - int(time.time()) print(f"Rate limited. Waiting {wait_time}s...") time.sleep(max(1, wait_time)) continue return response.json() raise Exception("Max retries exceeded") result = audit_with_retry('https://example.com', 'YOUR_API_KEY')
