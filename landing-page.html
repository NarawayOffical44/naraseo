Naraseo AI exposes a full REST API, MCP server, and OpenAPI spec. Paste your deploy URL into Claude Desktop, ChatGPT, or your own app and SEO intelligence is a native skill.

Claude Desktop / Cursor / Windsurf

MCP server built-in. Add one JSON block and every SEO tool becomes a native skill.

{
  "mcpServers": {
    "naraseo-ai": {
      "command": "npx",
      "args": [
        "mcp-remote",
        "https://YOUR-URL/mcp/sse"
      ]
    }
  }
}

Add to %APPDATA%\Claude\claude_desktop_config.json
